课6、函数(过程)

一、函数定义

函数-一组语句封装起来，一起执行

一般情况：函数定义，函数调用

为实现代码复用率，函数封装+函数调用

注意：(实践上就认为：函数封装=函数实现)，函数实现可包含函数定义

1.1、代码冗余

#避免代码重复，非常致命，使用函数；提高代码复用率

1.2、函数格式（函数名大驼峰命名）

a.定义- 返回值(类型) 函数名(参数列表) {函数体}
函数名-大驼峰命名，没有类型，每个单词的第一个字母大写
返回值(类型)-根据函数功能提供类型，如果没有返回值使用 void
参数列表-【类型 形参1[,类型 形参2]....]，可以没有参数

b.使用-函数名([实参列表])
如果没有参数，则不需要实参

c.注意
函数不能嵌套定义

1.3、vs单步功能

逐过程(F10)-单步步过，不进入函数内部，跳过函数实现
逐语句(F11)-单步入，进入函数内部，调试函数实现

二、声明和实现

函数声明(.h)+函数实现(.c)+调用(.c)

2.1、前向声明

当函数实现不在函数调用上方时，需要进行 函数声明
函数声明- 返回值 函数名(参数列表);

2.2、声明实现分开

#include<stdio.h> //库文件
#include “prints.h” //包含头文件，自己定义

声明 -一般放在 头文件 中，后缀名为 .h
实现 -一般放在 源文件 中，后缀名为.c
包含头文件 -在使用函数的 源文件 中使用语句 #include“XXXx h"
递进包含-当x.c包含了a.h而a.h包含了b.h则x.c也包含了 b.h

三、函数分类

自定义函数-开发者实现的函数，头文件包含一般使用双引号 #include "xxxx.h"
库函数- IDE 自带的函数，头文件包含一般使用尖括号 #include<xxxx.h>

课8、字符串处理函数

一、字符串

1、字符数组

字符数组是⼀个普通的数组，可以包含任意字符，不⼀定以 ‘\0’ 结尾。
例如： char arr[] = {'h', 'e', 'l', 'l', 'o'};

2、字符串

字符串：以字符数组的形式表示,以空字符‘/0’结尾
例如：char str[] = "hello"; // ⾃动以 ‘\0’ 结尾

二、字符串输入输出库函数

在 C 语⾔中，字符串的输⼊与输出通常使⽤标准输⼊输出函数 scanf 和 printf ，以及gets 和puts。

1、输入

scanf :

▪ 使⽤ %s 格式说明符来读取字符串。
▪ scanf 会在遇到空⽩字符（如空格、制表符或换⾏符）时停⽌读取。

gets :

• ⽤于读取⼀整⾏字符串，包括空格，直到遇到换⾏符。
• 不推荐使⽤ gets ，因为它不进⾏边界检查，可能导致缓冲区溢出。

getchar 的⽤法及注意事项:

• getchar 是 C 语⾔中的⼀个标准库函数，⽤于从标准输⼊（通常是键盘）读取下⼀个字符。

2、输出

printf :

• 使⽤ %s 格式说明符来输出字符串。

puts :

• ⽤于输出字符串，并在末尾⾃动添加换⾏符。

3、如何表⽰不可⻅字符

C 语⾔中，要在字符串中表⽰打印不出来的字符（如控制字符或特殊字符），通常使⽤转义序列。
换⾏符： \n
⽔平制表符： \t
垂直制表符： \v
回⻋符： \r
退格符： \b
警报（响铃）： \a
反斜杠： \\
单引号： \'
双引号： \"
问号： \?
空字符： \0
此外，可以使⽤⼋进制或⼗六进制转义序列来表⽰任意字符：
⼋进制： \ooo ，其中 ooo是⼀到三位的⼋进制数。例如，\101 表⽰字符 'A' ⼗六进制： \xhh，其中hh 是⼀到两位的⼗六进制数。例如， \x41 表⽰字符 'A'

4、 注意事项

返回类型

▪ getchar 返回⼀个 int 类型的值，⽽不是 char 。这是因为它需要能够表⽰所有可能的字符值以及特殊的 EOF（⽂件结束）标志。

EOF 检测

▪ getchar 在到达输⼊的末尾或发⽣读取错误时返回 EOF 。因此，通常在循环中使⽤getchar 时，需要检查其返回值是否为 EOF 。

缓冲区⾏为

▪ getchar 是⾏缓冲的，这意味着它在按下 Enter 键之前不会返回输⼊的字符。这可能会影响交互式程序的⾏为。
